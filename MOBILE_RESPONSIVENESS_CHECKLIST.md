## CSS Classes Used

### Responsive Typography
- `text-3xl md:text-4xl` - Responsive heading
- `text-base md:text-lg` - Responsive body text
- `text-xl md:text-2xl` - Responsive subheading

### Responsive Spacing
- `px-4 md:px-6` - Padding
- `py-12 md:py-20` - Vertical padding
- `pt-16 md:pt-24` - Top padding
- `gap-8 md:gap-12 lg:gap-16` - Grid gaps

### Responsive Layout
- `grid-cols-1 md:grid-cols-2 lg:grid-cols-4` - Grid columns
- `hidden md:flex` - Show on desktop only
- `md:hidden` - Hide on desktop
