# Prince Joshua N. Portfolio

Minimal, Ivy League-inspired personal portfolio built with Next.js 14 (App Router) and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build for production

```bash
npm run build
npm run start
```

## Customize

- Update social/profile placeholders marked with `TODO` comments in:
  - `components/Hero.js`
  - `components/Contact.js`
- Update metadata in `app/layout.js`.
- Add your CV file at `public/cv.pdf` or change the CV link in `components/Hero.js`.
